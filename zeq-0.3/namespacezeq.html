Detailed Description

Publish-subscribe classes for typed events.

A Publisher opens a listening port on the network, and publishes an Event on this port. It announces its publishing URI scheme for automatic discovery.

A Subscriber either explicitely subscribes to the publisher port, or uses automatic discovery to find publishers using the same URI scheme. Automatic discovery is implemented using zeroconf networking (avahi or Apple Bonjour).

The connection::Broker and connection::Service may be used to introduce a subscriber to a remote, not zeroconf visible, publisher.

An Event contains a strongly type, semantically defined message. Applications or groups of applications can define their own vocabulary.
